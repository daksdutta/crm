# Enterprise CRM Backend

Production-oriented Django CRM backend with:
- Multi-tenant architecture
- RBAC permissions
- Access request workflow
- Pipeline & Deal system
- Service layer architecture
- Repository pattern
- JWT authentication

## Tech Stack
- Django
- Django REST Framework
- PostgreSQL
- SimpleJWT
- drf-spectacular

## Features
- Organization-based tenant isolation
- Department & Group system
- Custom permission system
- Activity tracking
- Deal pipelines
- Access visibility controls

## Architecture
View → Service → Repository → Model
